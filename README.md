# SmartAttend – NFC Student Attendance Management System

## Overview
SmartAttend is a systems analysis and design project that proposes a digital attendance management system for universities. The system replaces manual paper registers with NFC-enabled student card scanning to improve accuracy, reduce fraud, and provide real-time attendance tracking.

---

## Problem Statement
Many universities still use manual attendance registers, which are inefficient, time-consuming, and prone to errors such as proxy attendance and lost records. This results in unreliable attendance data and poor student performance tracking.

---

## Proposed Solution
SmartAttend introduces an NFC-based attendance system where students tap their university cards to automatically register attendance. Lecturers can monitor attendance in real time, and administrators can generate reports for academic monitoring.

---

## Objectives
- Eliminate manual attendance registers
- Prevent proxy attendance
- Provide real-time attendance tracking
- Improve reporting and analytics
- Reduce administrative workload
